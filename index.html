FAQ

Kurz & knapp

Was ist ORP?
ORP steht für „Optimal Recognition Point“. Dabei wird ein Buchstabe pro Wort an einer stabilen Position hervorgehoben, sodass die Augen weniger springen.
Werden meine Daten hochgeladen?
Nein. Der Text wird ausschließlich lokal im Browser verarbeitet.

Datenschutz

Client-Side

Diese Seite verarbeitet Text nur lokal im Browser. Es werden keine Inhalte an einen Server übertragen.
